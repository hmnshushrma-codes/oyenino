🛒 The Prompt — Copy this entire thing
store-prompt.txt
Create a single HTML file e-commerce website called "Neo Market" with these features: PRODUCTS: - All products are stored in a JS array at the top of the file so I can edit them easily - Each product has: id, name, price, category, description, images (array of URLs), stock PAGES (use URL hash for routing): - Home page: product grid with cards, hero section, filter by category - Product page: full details, image gallery, quantity selector, related products - Cart page: list of items, quantities, total - Checkout page: form + order summary PRODUCT CARDS: - Clicking the card opens the product page - "Add to Cart" button only adds to cart, does not navigate - If item is already in cart, button shows "In Cart ✓" CART: - Store cart in localStorage under the key "neo_cart" - Cart icon in navbar shows item count - User can change quantity or remove items from cart page CHECKOUT: - Fields: Name, Email, Phone, Address, Notes - On submit, send cart data to Formspree (I will paste my Formspree URL) - Show success message, clear cart, redirect to home after 3 seconds CONFIG at top of file: - Store name, tagline, Formspree URL, currency symbol DESIGN: - Warm earthy colors, not generic blue - Serif font for headings, clean sans-serif for body - Mobile responsive - Smooth hover effects on cards Output: one complete .html file, no frameworks, no build tools.

📝 Prompt paste karne ke baad kya karo?

  • Store name change karo — "Neo Market" ki jagah apna naam daalo
  • Products mein apne items daalo — naam, price, photo URL, category
  • Formspree URL — formspree.io pe free account banao, URL paste karo
  • Design colors bhi change karwa sakte ho — prompt mein bol do "use pink and gold" ya jo chahiye

📷 Product photos kaise lagayein?

  • Imgur.com pe upload karo — free, direct link milega
  • ImgBB.com — free, fast, no signup needed
  • Tip: 600x600px square photos sabse acchi dikhti hain

🚀 Live kaise karo?

  • Easiest: Netlify Drop — file drag karo, 10 sec mein live
  • Also free: GitHub Pages, Cloudflare Pages, Vercel
  • Instagram bio mein link daalo — done! Customers direct order karenge
