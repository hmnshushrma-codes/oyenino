Ready-to-use store template

Ek HTML file. Products, cart, checkout, email orders. Sab kuch. Kholke products change karo aur live ho jao.

✍️ Step 1 — Config change karo

⚙️ Top of file mein 4 cheezein change karo:

  • storeName — Apni dukaan ka naam, e.g. "Priya's Boutique"
  • tagline — Ek line description
  • currency — ₹ already set hai
  • formspreeURL — formspree.io pe free account banao, URL paste karo
config section
var CONFIG = { storeName: 'Priya\'s Boutique', tagline: 'Handpicked fashion for every occasion', currency: '\u20B9', formspreeURL: 'https://formspree.io/f/YOUR_ID', };
🛒 Step 2 — Products add karo

📦 Har product mein ye daalo:

  • name — Product ka naam
  • price — Sirf number, e.g. 499
  • category — Filter ke liye, e.g. "Earrings"
  • images — Photo URLs (Imgur, ImgBB, ya koi bhi link)
  • stock — Kitne available hain (0 = sold out)
products array
{ id: 1, name: 'Gold Jhumka Set', price: 499, category: 'Earrings', description: 'Beautiful traditional jhumkas...', images: ['https://your-image-url.jpg'], stock: 20, }
🚀 Step 3 — Live karo

🌐 Hosting options (sab free):

  • Easiest: Netlify Drop — file drag karo, 10 sec mein live
  • Also free: GitHub Pages, Cloudflare Pages, Vercel
  • Custom domain: ₹500/year mein mil jaata hai (optional)
  • Final step: Instagram bio mein URL daalo — done!

📷 Product photos kaise upload karein?

  • Imgur.com — Free, direct link milega
  • ImgBB.com — Free, fast, no signup
  • Tip: 600x600px square photos sabse acchi dikhti hain
